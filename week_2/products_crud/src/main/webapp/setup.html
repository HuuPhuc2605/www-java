Hướng dẫn cài đặt CRUD Products

Bước 1: Tạo Database và Bảng

Mở MySQL Workbench hoặc Command Line và thực hiện các lệnh sau:

CREATE DATABASE shop_db;
USE shop_db;

CREATE TABLE products (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(255) NOT NULL,
    price DOUBLE NOT NULL,
    url_image VARCHAR(500)
);

INSERT INTO products (name, price, url_image) VALUES

Bước 2: Kiểm tra Cấu hình MySQL

Mở tệp ProductDAO.java và kiểm tra:

private static final String URL = "jdbc:mysql://localhost:3307/shop_db";
private static final String USER = "root";
private static final String PASS = "";

Đảm bảo:

  • Port MySQL là 3307 (hoặc thay đổi nếu cần)
  • Username là root
  • Password trống "" (nếu khác, hãy cập nhật)

Bước 4: Truy cập trang quản lý sản phẩm

Vào Danh sách Sản phẩm

Lỗi thường gặp:

❌ "Chưa có sản phẩm nào"

  • Kiểm tra database đã được tạo chưa
  • Kiểm tra dữ liệu mẫu đã được chèn chưa
  • Xem Kiểm tra kết nối

❌ "Connection failed"

  • Kiểm tra MySQL có đang chạy không
  • Kiểm tra port MySQL (mặc định 3306, dự án này dùng 3307)
  • Kiểm tra username/password trong ProductDAO.java

❌ "Table 'products' doesn't exist"

  • Kiểm tra database shop_db đã được tạo chưa
  • Chạy lại câu lệnh CREATE TABLE ở Bước 1

MySQL Port 3307 là gì?

Dự án này sử dụng port 3307 thay vì port mặc định 3306. Nếu MySQL của bạn chạy trên port khác:

1. Mở ProductDAO.java

2. Thay đổi dòng:

private static final String URL = "jdbc:mysql://localhost:3307/shop_db";

thành:

private static final String URL = "jdbc:mysql://localhost:3306/shop_db";

3. Compile lại dự án
